--------------------------------------------------------------------
Readme for Mike's Process Blocker (MPB) v.1.2
--------------------------------------------------------------------

Features:
	- blocks existing programms
	- configurable blockfile
	- logging
	- automatically creates a block- and a logfile for each user
	- hotkeys:	CTRL + ALT + s starts/stops blocking
			CTRL + ALT + h hides/shows the window

	NOTE: Please use the program for configuring the blockfile!
